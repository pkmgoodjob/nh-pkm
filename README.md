# 🏛️ 전국 농협 교육원 통합 관리 앱 (Master Template)

이 프로젝트는 전국 농협 교육원 담당자들이 각 교육원의 상황에 맞춰 **코딩 없이 1분 만에** 자신만의 앱을 구축할 수 있도록 제작된 마스터 템플릿입니다.

## 🚀 1분 만에 앱 배포하기

아래 버튼을 누르면 담당자님의 Vercel 계정으로 이 코드가 복제되며 즉시 배포가 시작됩니다.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fpkmgoodjob%2Fnh-pkm&env=VITE_SUPABASE_URL,VITE_SUPABASE_ANON_KEY,VITE_ADMIN_PASSWORD)

---

## 🛠️ 준비 사항 (배포 버튼 클릭 시 필요)

버튼을 누르기 전, 아래 정보를 미리 준비해 주세요:
1. **Supabase 정보:** [Supabase](https://supabase.com) 프로젝트의 `URL`과 `Anon Key`.
2. **관리자 비밀번호:** 앱 설정 페이지에 접속할 때 사용할 비밀번호 (본인이 정함).

## 📖 사용 및 관리 방법

1. **앱 접속:** 배포가 완료되면 부여된 인터넷 주소로 접속합니다.
2. **설정 변경:** 주소 뒤에 `/admin`을 붙여 관리자 페이지에 접속합니다. (예: `https://your-app.vercel.app/admin`)
3. **콘텐츠 수정:** 교육원 이름, 슬로건, 시설 안내, 조감도 등을 입력하면 앱에 실시간으로 반영됩니다.
